# Changelog проекта «Печатает Максим»

Все значимые изменения проекта фиксируются в этом документе.

Формат основан на принципах Keep a Changelog. Записи группируются по версиям или рабочим этапам и категориям изменений.

## Unreleased

### Added

- Главная страница и Studio формируют скачиваемый текстовый файл заявки.
- Добавлены доступные роли, подписи и клавиатурное управление диалогами Studio.
- Добавлены smoke-тесты главной страницы, заявки и реального JPEG.
- Roadmap дополнен поэтапным планом развития и критериями готовности.

### Changed

- Главная страница больше не сообщает о доставке заявки без реальной отправки.
- README и архитектурная документация приведены в соответствие текущему runtime.

### Fixed

- Устранён горизонтальный overflow главной страницы на мобильной ширине.
- Финальная кнопка Studio теперь выполняет заявленное действие.

### Removed

- Ничего не удалено.
