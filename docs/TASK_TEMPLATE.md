# Шаблон задачи

Скопируйте блок ниже в ChatGPT, GitHub Issue или сообщение другому участнику.

```text
Номер: [например, STUDIO-022]
Название:

Раздел:
[Главная / 01 Создать / 02 Настроить / 03 Получить / инфраструктура]

Устройство и размер:
[мобильное 390 px / desktop 1440 px / оба]

Текущее поведение:

Ожидаемый результат:

Не менять:

Материалы:
[снимок, видео, ссылка или отсутствуют]

Критерии готовности:
- ...
- ...

Режим:
[обсудить / реализовать локально / показать / публиковать]
```

## Короткая версия

Для небольшой визуальной задачи достаточно:

```text
Номер:
Раздел:
Устройство:
Проблема:
Ожидается:
Не менять:
Режим:
```

## Команды режима

- `Обсудим` — анализ без изменений.
- `Реализуй` — локальное изменение и проверки без commit.
- `Покажи` — запуск и визуальная проверка.
- `Публикуй` — документация, commit, PR, merge и deployment.
