# Шрифты Studio

Локальные WOFF2-файлы подготовлены из согласованной подборки проекта:

- Manrope
- Unbounded
- Comfortaa
- Play
- Yeseva One
- Commissioner
- Dela Gothic One
- Forum
- IBM Plex Sans
- PT Sans
- PT Serif
- Pacifico
- Playfair Display

Файлы содержат латиницу, кириллицу, цифры и основные типографские символы.
Исходные гарнитуры распространяются по лицензии SIL Open Font License 1.1.
Тексты лицензий сохранены в каталоге `licenses`.
