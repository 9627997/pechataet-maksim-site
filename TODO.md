# Актуальные задачи

Подробный порядок и критерии готовности находятся в [docs/ROADMAP.md](docs/ROADMAP.md).

## P0

- объединить и выпустить текущие проверенные изменения Studio;
- подключить реальную защищённую доставку заявки;
- добавить лимиты и строгую проверку загружаемых файлов.

## P1

- декомпозировать монолит Studio без изменения поведения;
- завершить WCAG-аудит и добавить axe/WebKit в CI;
- внедрить визуальные regression-тесты и performance budget.

## P2

- сформировать полноценный производственный пакет заказа;
- добавить privacy-friendly аналитику доставленных заявок;
- развивать SEO/GEO только на основе подтверждённых продуктов и данных.
